# ProductHunt Launch — Asset Checklist

Pre-launch checklist for submitting the MCP Security Platform to ProductHunt.

---

## Required Assets

| Asset | Dimensions | Status | Notes |
|-------|-----------|--------|-------|
| Logo | 240x240 PNG | ❌ Needed | Square logo, no text overflow. Use the KryptosAI mark or a platform icon. |
| Hero image | 1270x760 PNG | ❌ Needed | Main product image. Show dashboard, demo frame, or the proxy-diagram from README. Should include the tagline. |
| Demo GIF | Any | ✅ Done | `docs/demo.gif` — terminal recording showing `init`, `proxy`, and a blocked call. Keep under 3MB. |
| Tagline | Text | ✅ Done | "Scan Before You Trust. Enforce at Runtime." |
| First comment | Text | ✅ Done | Introduce the platform, explain the problem it solves, and link to GitHub repos. |
| Social preview SVG | 1200x630 SVG | ✅ Done | `content/social-preview.svg` — dark gradient with logo, tagline, and github.com/KryptosAI URL. |
| Topics / tags | List | ✅ Done | MCP, Security, Developer Tools, Open Source, AI, Proxy, CLI |

---

## Additional Assets (Recommended)

| Asset | Purpose |
|-------|---------|
| Feature screenshots (3-4) | Show dashboard, policy YAML, blocked request log, comparison table |
| Thumbnail | 240x240 — same as logo, but centered for thumbnail display |
| Social preview image | ✅ Done — 1200x630, used when sharing the PH link on Twitter/LinkedIn (see `social-preview.svg`) |

---

## Maker Checklist

- [ ] ProductHunt account created
- [ ] Maker profile complete (name, avatar, bio, Twitter/GitHub links)
- [ ] Product name finalized: "MCP Security Platform" or individual tools?
- [ ] Product description written (2-3 sentences)
- [ ] GitHub repos linked
- [ ] Website linked (`kryptosai.github.io`)
- [ ] First comment ready (markdown, links to repos, docs, and Discord)
- [ ] Launch date selected (Tuesday-Thursday recommended)
- [ ] Maker community joined (PH Makers Discord, relevant subreddits)

---

## Launch Day Checklist

- [ ] Post scheduled or submitted at 12:01 AM PST
- [ ] First comment posted immediately
- [ ] Social media posts queued (Twitter, LinkedIn, Reddit, Discord announcement)
- [ ] Team available in first 4 hours to respond to comments
- [ ] Upvotes ready from early supporters (no ask-and-upvote coordination — organic only)
- [ ] PH page URL shared in project READMEs and landing page

---

## Post-Launch

- [ ] Thank-you comment posted after day 1
- [ ] Respond to all feedback comments
- [ ] Add "Featured on ProductHunt" badge to READMEs
- [ ] Review analytics (upvotes, comments, referral traffic)
- [ ] Write a launch retrospective for the blog

---
